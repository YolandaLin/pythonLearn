# 期末專題規格

## 專題目標

完成一個可以展示的 Python 小作品。題目可以來自生活、學校、社團、科展或個人興趣。

## 題目範例

- 讀書時間紀錄與圖表分析
- 班級問卷統計工具
- 單字複習小測驗
- 校園活動報名資料整理
- 天氣資料查詢與穿搭提醒
- 文字冒險或猜謎遊戲
- 個人記帳與分類統計

## 必備內容

1. README.md：作品介紹、安裝方式、執行方式、範例輸出。
2. 至少一個 .py 主程式。
3. 至少三組測試資料或操作截圖。
4. inal_report.md：開發過程、遇到的問題、下一版規劃。
