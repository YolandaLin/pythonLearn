## 常見錯誤

### 1. images 資料夾不存在

如果儲存圖片時出錯，先確認 `images` 資料夾是否存在。

### 2. 中文顯示問題

matplotlib 在某些電腦上可能無法正確顯示中文。可以先用英文標題與欄位完成作業，再請老師協助處理中文字型。

### 3. 忘記儲存圖檔

只寫 `plt.show()` 可能只會顯示視窗。請記得：

```python
plt.savefig("images/sleep_line.png")
```

### 4. 圖表沒有軸標籤

別人看圖時需要知道 x 軸和 y 軸代表什麼。沒有標籤的圖表不算清楚。

## 修正方向

1. 先用固定資料畫成功，再讀 CSV。
2. 先畫一張圖，不要兩張一起 debug。
3. 每次產圖後打開 png 檢查是否真的看得懂。
4. 在 README 說明圖檔代表什麼。
