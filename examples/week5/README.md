## 常見錯誤

### 1. 以為 strip 會移除所有空白

```python
"I love Python".strip()
```

中間的空白不會消失。`strip()` 只處理前後空白。

### 2. 空字串 split 的結果

```python
"".split()
```

結果是空 list：`[]`。所以單字計數要考慮空輸入。

### 3. 大小寫造成關鍵字找不到

`"python" in "I love Python"` 是 `False`，因為大小寫不同。可以先：

```python
message = message.lower()
```

再判斷小寫關鍵字。

### 4. 中文和英文的「字」概念不同

`len("小明")` 是 2，因為有兩個中文字。英文句子的單字數通常要用 `split()`，不是 `len()`。

## 修正方向

1. 文字輸入先想要不要 `strip()`。
2. 要做不分大小寫搜尋時，先統一轉成小寫。
3. 測試資料要包含空白、大小寫、沒有關鍵字。
4. 不要急著用很長的句子，先用短字串確認結果。
