## 常見錯誤

- 只讓程式在一組範例輸入下成功，沒有測邊界情況。
- 變數名稱太籠統，例如 、、x，導致一週後看不懂。
- 沒有在 README 寫如何執行程式。
- 把錯誤訊息只看最後一行，沒有回頭看檔名與行號。
