# 作業總覽

每一週的作業都放在 `weekN/README.md`。建議學生用自己的 GitHub repo 繳交，不直接改這份課綱 repo。

完全零基礎學生請先讀：

- [電腦基礎補強](../resources/computer-basics.md)
- [補充影片清單](../resources/videos.md)

作業類型通常包含：

1. 必做程式題：確認本週語法能用。
2. 簡答題：確認觀念有理解。
3. 挑戰題：給已經完成基礎題的學生延伸。
4. 學習日誌：記錄卡關與修正過程。

交作業前請先看對應的 `examples/weekN/README.md` 做自我檢查。
