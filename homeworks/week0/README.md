# 第 0 週作業：開課準備與學習方法

## 本週目標

建立學習節奏、環境與提問習慣。

## 本週主題

- Python 可以做什麼
- 安裝 Python 與 VS Code
- 使用終端機執行程式
- GitHub 與學習紀錄
- 如何拆問題、查資料、問問題

## 必做作業

- 安裝 Python、VS Code 與 Git，截圖或文字記錄版本號。
- 建立 learning-log.md，寫下本週學到的三件事與一個問題。
- 用自己的話解釋「程式是一行一行被執行的指令」。

## 繳交內容

請建立一個 week0 資料夾，至少包含：

- 本週 Python 程式檔。
- README.md：說明如何執行、測試資料與執行結果。
- learning-log.md：本週學習日誌。
