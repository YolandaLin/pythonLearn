# 第 0 週作業：電腦基礎、環境準備與學習方法

## 本週目標

先補齊檔案、資料夾、終端機與程式執行的基本概念，再建立 Python 開發環境與學習紀錄習慣。

## 本週主題

- 檔案、資料夾、副檔名與路徑
- 終端機的目前位置與基本指令
- Python 程式如何被執行
- 安裝 Python、VS Code 與 Git
- GitHub 與學習紀錄
- 如何拆問題、查資料、問問題

- [補充影片清單：課前必看與第 0 週](../../resources/videos.md)

## 必做作業

- 截圖或文字記錄 Python、VS Code、Git 的版本號。
- 建立 `hello.py`，內容印出自己的名字與一句本週目標。
- 用終端機執行 `python hello.py`，並把執行結果貼到 README。
- 建立 `learning-log.md`，寫下本週學到的三件事與一個問題。
- 用自己的話解釋「終端機目前位置」和「檔案路徑」是什麼。

## 繳交內容

請建立一個 `week0` 資料夾，至少包含：

- `hello.py`
- `README.md`：記錄版本號、執行指令與執行結果。
- `learning-log.md`：本週學習日誌。

## 自我檢測

- [ ] 能顯示檔案副檔名，分辨 `hello.py` 與 `hello.py.txt`。
- [ ] 能在終端機執行 `python --version`。
- [ ] 能用 `cd` 進入 `hello.py` 所在資料夾。
- [ ] 能執行 `python hello.py`。
- [ ] 能從錯誤訊息找到檔名與行號。
- [ ] 知道如何用 Git commit 一次變更。

## 挑戰題

故意把檔名改成錯誤路徑或輸入不存在的檔名，觀察錯誤訊息。請在 README 寫下：

1. 你輸入了什麼指令。
2. 電腦回傳什麼錯誤。
3. 你如何判斷問題出在哪裡。
