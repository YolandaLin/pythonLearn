# 第 2 週作業：條件判斷

## 本週目標

## 本週主題

- if/elif/else
- 比較運算
- 布林值
- 巢狀判斷
- 邊界條件

## 必做作業

- 寫 grade.py：輸入分數，輸出 A/B/C/D/F。
- 寫 ticket.py：依年齡輸出票價類別。
- 列出 grade.py 至少 5 組測試資料，包含邊界值。
