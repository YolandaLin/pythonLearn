# 第 2 週作業：條件判斷

## 本週目標

讓程式可以根據不同輸入做出不同回應。先從「是或否」的單一判斷開始，再進到多種結果的判斷。

這一週的重點不是背語法，而是看懂：

```python
if 條件成立:
    做這件事
else:
    不成立就做另一件事
```

## 本週主題

- `if`：如果條件成立
- `else`：否則
- `elif`：否則如果
- 比較運算：`==`、`!=`、`>`、`>=`、`<`、`<=`
- 布林值：`True` / `False`
- 邊界值：剛好等於門檻的情況

## 範例 1：是否成年

```python
age_text = input("你今年幾歲？")
age = int(age_text)

if age >= 18:
    print("你已經成年")
else:
    print("你還未成年")
```

請注意：`>= 18` 包含剛好 18 歲。

## 範例 2：密碼檢查

```python
password = input("請輸入密碼：")

if password == "python123":
    print("登入成功")
else:
    print("密碼錯誤")
```

請注意：判斷是否相等要用 `==`，不是 `=`。
