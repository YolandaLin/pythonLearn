# 第 1 週作業：輸出、輸入與變數

## 本週目標

先讓學生成功寫出會互動的小程式，再理解「資料可以放進變數」以及 `input()` 拿到的是文字。

這一週不要急著寫複雜公式。重點是：看得懂程式從上到下怎麼跑、能自己改文字、能用終端機執行 `.py` 檔。

## 本週主題

- `print()`：把文字印到畫面上
- `input()`：讓使用者輸入資料
- 變數：替資料取名字
- 字串：文字資料
- 整數：可以計算的數字
- `int()`：把文字轉成整數
- f-string：把變數放進句子裡

## 寫作業前先做

請先確認你能做到：


## 範例 1：印出文字

建立 `hello.py`：

```python
print("哈囉，Python")
print("這是我的第一週作業")
print("我今天要學會變數")
```

執行：

```powershell
python hello.py
```

## 範例 2：輸入名字

建立 `ask_name.py`：

```python
name = input("你的名字是什麼？")
print(f"你好，{name}")
```

請注意：`name` 是變數，裡面放的是使用者輸入的文字。

### hw3：明年幾歲

建立 `next_age.py`，讓使用者輸入今年幾歲，輸出明年幾歲。

提示：`input()` 拿到的是文字。如果要做加法，要先用 `int()` 轉成整數。

```python

age = int(age_text)
print(f"你明年會是 {age + 1} 歲")
```

## 挑戰題：BMI 計算機

建立 `bmi.py`，輸入身高與體重，輸出 BMI。

這題會用到小數與公式，完全零基礎學生可以先跳過，等老師帶著做。

公式：

```text
BMI = 體重公斤 / (身高公尺 * 身高公尺)
```

提示：身高和體重要用 `float()` 轉成小數。
