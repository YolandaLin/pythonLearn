# 第 1 週作業：變數、輸入與輸出

## 本週目標

能把生活中的資料放進變數並產生輸出。

## 本週主題

- print 與 input
- 變數命名
- 字串、整數、浮點數
- 型別轉換
- f-string

## 必做作業

- 寫 bmi.py：輸入身高體重，輸出 BMI 與一句提醒。
- 寫 intro.py：輸入姓名、年級、興趣，輸出完整自我介紹。
- 簡答：字串 '18' 和數字 18 有什麼不同？
