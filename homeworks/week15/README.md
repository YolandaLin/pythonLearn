# 第 15 週作業：小型專題規劃

## 本週目標

把前面學到的能力整合成一個可展示作品。

## 本週主題

- 需求拆解
- 功能清單
- 資料設計
- 任務切分
- README 寫法

## 必做作業

- 寫 project_proposal.md：題目、使用者、功能、資料來源。
- 完成至少一個可執行的核心功能。
- 建立專題 README，說明如何執行。

## 繳交內容

請建立一個 week15 資料夾，至少包含：

- 本週 Python 程式檔。
- README.md：說明如何執行、測試資料與執行結果。
- learning-log.md：本週學習日誌。

## 自我檢測

- [ ] 能說清楚作品要解決的問題
- [ ] 能把功能切成 must/should/could
- [ ] 能讓同學照 README 跑起作品

## 挑戰題

把本週作品加上一個「使用者輸入錯誤時不會直接當掉」的處理，並在 README 寫出你測過哪些錯誤輸入。
