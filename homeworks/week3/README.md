# 第 3 週作業：迴圈與重複工作

## 本週目標

學會讓程式重複做事。高中生常見卡點不是語法，而是不知道「第幾圈迴圈時變數是多少」。這週請一邊寫程式，一邊在紙上追蹤每一圈的變化。

## 本週主題

- `for`：已經知道要重複幾次時使用
- `range()`：產生一串數字
- `while`：不知道要重複幾次，但知道停止條件時使用
- 累加器：把每一圈的結果加起來
- `break`：提早結束迴圈

## 範例 1：印出 1 到 5

```python
for number in range(1, 6):
    print(number)
```

請注意：`range(1, 6)` 會產生 1、2、3、4、5，不包含 6。

## 範例 2：累加總和

```python
total = 0

for number in range(1, 6):
    total = total + number

print(total)
```

請用紙寫下每一圈的 `number` 和 `total`。
