# 第 4 週作業：函式與問題拆解

## 本週目標

學會把大問題拆成可測試的小函式。

## 本週主題

- def
- 參數與回傳值
- 區域變數
- 早期 return
- 簡單測試

## 必做作業

- 寫 converter.py：包含 c_to_f、f_to_c、km_to_mile 三個函式。
- 寫 is_even、is_prime、max_of_three 三個函式。
- 簡答：print 和 return 的差異是什麼？
