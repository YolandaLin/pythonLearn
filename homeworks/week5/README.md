# 第 5 週作業：字串處理

## 本週目標

能處理文字資料，做出簡單文字分析。

## 本週主題

- 索引與切片
- len
- split/join
- lower/upper/strip
- 字串搜尋

## 必做作業

- 寫 reverse_text.py：不使用 [::-1]，反轉輸入字串。
- 寫 word_count.py：統計一段文字的字數。
- 寫 keyword.py：判斷留言是否包含指定關鍵字。
