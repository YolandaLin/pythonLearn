# 第 5 週作業：字串處理

## 本週目標

學會處理文字資料。Python 不只可以算數字，也可以整理姓名、留言、文章、關鍵字。這週重點是把文字拆開、清理、搜尋與重新組合。

## 本週主題

- 索引：取得某個位置的字元
- 切片：取得一段文字
- `len()`：計算長度
- `strip()`：去掉前後空白
- `lower()` / `upper()`：大小寫轉換
- `split()`：把文字拆成 list
- `in`：判斷是否包含某段文字

## 範例 1：清理名字

```python
name = input("請輸入名字：")
name = name.strip()
print(f"你好，{name}")
```

## 範例 2：關鍵字搜尋

```python
message = input("請輸入留言：")

if "考試" in message:
    print("這則留言提到考試")
else:
    print("這則留言沒有提到考試")
```
